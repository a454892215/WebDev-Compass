# WebDev Compass

本项目用于建立一个 **Web 开发知识库**，帮助学习者从入门到进阶，系统掌握 Web 开发的核心知识与工程能力。

## 主要内容

- **[Web 工程师知识地图（KNOWLEDGE_MAP.md）](./KNOWLEDGE_MAP.md)**：从入门到高级的完整知识体系（阶段 0–15），知识点采用四级序号（如 `0.1.1`、`1.1.2.3`），笔记与示例文件与序号对齐。
- **路线图对应的知识体系笔记**：围绕每个阶段整理的概念、原理与实践要点，便于查阅与复习。
- **完整代码示例**：与知识点配套的可运行示例代码，覆盖常见场景与典型问题。
- **主流框架工程示例**：包含 Web 开发主流框架的工程化示例项目，帮助理解真实项目结构与开发流程。

## 学习目标

完成本项目的学习后，你将能够：

- 建立全面且深入的 Web 开发知识体系与实践经验；
- 达到 **高级 Web 开发工程师** 的能力水平；
- 具备 **独立承担复杂 Web 项目** 的能力，涵盖架构设计、生产级工程治理与团队协作；
- 能够应对企业级 Web 开发需求：页面与 SPA 开发、接口对接与 BFF、状态管理、工程化构建、性能与安全治理、监控与部署、设计系统与平台化建设等。
