# WebDev Compass

本项目用于建立一个 **Web 开发知识库**，帮助学习者从入门到进阶，系统掌握 Web 开发的核心知识与工程能力。

## 主要内容

- **[知识库地图（KNOWLEDGE_MAP.md）](./KNOWLEDGE_MAP.md)**：按阶段整理的 Web 开发完整知识点清单，含学习里程碑与移动端经验对照。
- **Web 开发学习路线图**：从基础到框架到工程化的学习路径与阶段目标。
- **路线图对应的知识体系笔记**：围绕每个阶段整理的概念、原理与实践要点，便于查阅与复习。
- **完整代码示例**：与知识点配套的可运行示例代码，覆盖常见场景与典型问题。
- **主流框架工程示例**：包含 Web 开发主流框架的工程化示例项目，帮助理解真实项目结构与开发流程。

## 学习目标

完成本项目的学习后，你将能够：

- 建立较为完善的 Web 开发知识体系与实践经验；
- 达到 **中级 Web 开发工程师** 的能力水平；
- 具备 **独立完成 Web 开发** 的能力；
- 能够独立应对大部分企业常见的 Web 开发需求（页面开发、接口对接、状态管理、工程化构建、性能优化与基础部署等）。
