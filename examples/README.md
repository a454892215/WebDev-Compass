# 示例代码目录

本目录存放与 [notes/](../notes/) 中知识点配套的**可运行示例**。

## 目录结构

按学习阶段编号，与 `notes/` 目录对应：

```
examples/
├── 00-environment/
│   └── 0.1-what-is-web-app/
│       ├── request-response-demo.html
│       ├── spa-vs-mpa-demo.html
│       └── same-origin-demo.html
└── ...
```

## 如何运行

1. 直接用浏览器打开 `.html` 文件，或
2. 使用 VS Code **Live Server** 插件启动本地服务器（推荐，部分示例需要 `http://` 协议）
