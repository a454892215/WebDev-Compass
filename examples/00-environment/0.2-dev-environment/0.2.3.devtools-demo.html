Chrome DevTools 六项面板练习

对应笔记 0.2.3 · 请用 Live Server 以 http:// 打开，按 F12 开始

开始前：打开 DevTools → Network 勾选 Disable cache； Console 可勾选 Preserve log（跳转场景用，本页可选）。

Elements1. 元素与样式

任务：用选择器 (Cmd+Shift+C) 选中下方橙色方块 → 在 Styles 里把 background 改成 #4caf50 → 在 :hov 勾选 :hover 预览悬停色。

方块

Console2. 控制台

任务：点击下方按钮，在 Console 查看 log / table；在 Console 输入 $0（需先在 Elements 选中元素）。

Console 输出也会 mirror 到这里（可选对照）

Network3. 网络请求

任务：打开 Network 面板 → 点击「获取用户」→ 点开请求查看 Headers / Preview / Timing。

    Sources4. 源码与断点

    任务：打开 Sources → 找到本页脚本 → 在 debuggerDemo 函数内行号处设断点 → 点击按钮，观察 Scope 与 Call Stack。

    Performance5. 性能录制

    任务：Performance → Record → 点击「模拟重计算」→ Stop → 在 Main 线程查看长任务（黄色/红色区域）。

    Application6. 存储

    任务：点击「写入 localStorage」→ Application → Local Storage → 本页 origin → 查看键 compass-demo-theme；再点「清除」验证。
