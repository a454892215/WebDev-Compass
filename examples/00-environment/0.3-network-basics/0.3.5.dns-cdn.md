# 0.3.5 DNS / CDN 观察清单

对应笔记：[0.3.5 DNS 与 CDN](../../../notes/00-environment/0.3-network-basics/0.3.5.dns.md)

## A. DNS

```bash
# macOS / Linux 常见
dig example.com +short
dig www.cloudflare.com CNAME +short
```

观察：

1. `A` / `AAAA` 返回的 IP 是否随运营商变化（可对比手机热点 vs 家庭宽带，若条件允许）。
2. DevTools → Network → 某文档请求 → **Timing** → 是否有 **DNS Lookup** 耗时（连接复用后常为 0）。

## B. CDN

1. 打开使用 CDN 的站点（许多文档站、开源站静态资源在 `cdn.` / `*.cloudfront.net` / `*.akamai*` 等域名下）。
2. Network 中筛选 JS/CSS，看 **Remote Address** 与 **Response Headers** 中是否有 `cf-cache-status`、`x-cache`、`age` 等缓存命中线索（因厂商而异）。
3. 对比同源静态资源 vs 第三方 CDN 域名：是否多一次 DNS + 连接。

## C. 前端提示

在 HTML 中对关键第三方源可加：

```html
<link rel="dns-prefetch" href="https://cdn.example.com" />
<link rel="preconnect" href="https://cdn.example.com" crossorigin />
```

本仓库不强制运行服务；把观察结果记在笔记旁即可。
