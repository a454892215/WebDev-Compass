# 0.3.3 HTTP 版本观察清单

对应笔记：[0.3.3 HTTP/1.1 · HTTP/2 · HTTP/3](../../../notes/00-environment/0.3-network-basics/0.3.3.http-http-http.md)

本示例以 **观察真实网站** 为主（无需本地服务）。

## 步骤

1. 打开 Chrome，访问任意主流站点（如 `https://www.cloudflare.com` 或你常用的文档站）。
2. `F12` → **Network**，刷新页面。
3. 表头右键 → 勾选 **Protocol**。
4. 查看若干资源的 Protocol 列：

| 值 | 含义 |
| --- | --- |
| `h3` | HTTP/3 |
| `h2` | HTTP/2 |
| `http/1.1` | HTTP/1.1 |
| `data` / `blob` | 非网络协议（本地） |

5. 点开某一请求 → **Timing**，理解排队、DNS、连接、等待、下载与协议无关的成本仍在。
6. 再打开 `http://example.com`（若仍提供 HTTP），对比地址栏「不安全」提示与 HTTPS 站点。

## 要点回顾

- 业务代码一般 **不必** 为 h2/h3 改 API。
- 本地 `localhost` 开发常见 `http/1.1`，属正常。
- 性能仍依赖缓存、体积与关键请求数量。
