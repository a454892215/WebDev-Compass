# 知识笔记目录

本目录存放 [KNOWLEDGE_MAP.md](../KNOWLEDGE_MAP.md) 中各知识点对应的**详细笔记**。配套可运行代码见 [examples/](../examples/) 目录。

## 目录结构

按学习阶段编号组织，与知识库地图中的阶段一一对应：

```
notes/
├── 00-environment/          # 阶段 0：环境与心智模型
│   ├── 0.1-what-is-web-app/
│   ├── 0.2-dev-environment/     （待补充）
│   └── 0.3-network-basics/      （待补充）
├── 01-html/                 # 阶段 1：HTML（待补充）
├── 02-css/                  # 阶段 1：CSS（待补充）
├── 03-javascript/           # 阶段 1–2：JavaScript（待补充）
└── ...
```

## 如何使用

1. 在 [KNOWLEDGE_MAP.md](../KNOWLEDGE_MAP.md) 中点击知识点链接，跳转到对应笔记。
2. 阅读笔记中的概念说明、图示与 **📱 移动端对照**。
3. 按笔记底部的「实践」指引，在 `examples/` 中运行配套示例代码。
4. 学完后回到知识库地图，将对应项勾选为 `[x]`。
